## 👀 What Project is It?

우리의 목표는 의료 분야에서 서로를 빠르고 정확하게 연결하여 의료 서비스의 품질을 향상시키는 데 기여하는 것입니다.✍️ 
이를 통해 의료 현장의 효율성을 증대시키고, 더 나은 의료시스템을 만드는 것에 기여하는 것 입니다.🎉

<br/>
